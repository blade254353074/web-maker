密码修改

  1. 密码不得包含全部或部分用户帐户名。部分帐户名是指三个或三个以上两端用“空白”（空格、制表符、回车符等）或任何以下字符分隔的连续字母数字字符：逗号 (,)、句点 (.)、连字符 (-)、下划线 (_) 或数字符号 (#)。
  2. 密码长度(需要符合Default Domain Policy中密码策略要求的最小长度)。
  3. 密码必须包含以下四类字符中的三类：
    1. 拉丁文大写字母 (A – Z)
    2. 拉丁文小写字母 (a – z)
    3. 10 个基本数字 (0 – 9)
    4. 非字母数字字符，如感叹号 (!)、美元符号 ($)、数字符号 (#) 或百分号 (%)。
